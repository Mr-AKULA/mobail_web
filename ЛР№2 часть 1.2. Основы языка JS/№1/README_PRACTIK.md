## Задание 1: Ex1.html

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Задание 1 - Встроенный скрипт</title>
</head>
<body>
    <h1>Задание 1</h1>
    
    <p>Это первый абзац с произвольным текстом. Здесь может быть любая информация, которую вы хотите отобразить на веб-странице.</p>
    
    <script>
        // Вывод приветственного сообщения в диалоговом окне
        alert("Добро пожаловать на нашу страницу!");
        
        // Вывод приветственного сообщения в окно браузера
        document.write("<p style='color: blue; font-weight: bold;'>Приветствуем вас на нашей странице!</p>");
    </script>
    
    <p>Это второй абзац с произвольным текстом. После закрытия диалогового окна вы увидите приветственное сообщение, добавленное непосредственно в документ.</p>
</body>
</html>
```

## Задание 2: Ex2.html и main.js

**Ex2.html:**
```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Задание 2 - Внешний скрипт</title>
</head>
<body>
    <h1>Задание 2</h1>
    
    <p>Это первый абзац с произвольным текстом. В этом задании JavaScript код вынесен в отдельный файл.</p>
    
    <!-- Подключение внешнего скриптового файла -->
    <script src="main.js"></script>
    
    <p>Это второй абзац с произвольным текстом. Скрипт был выполнен из внешнего файла main.js.</p>
</body>
</html>
```

**main.js:**
```javascript
// Вывод приветственного сообщения в диалоговом окне
alert("Приветствие из внешнего файла! Добро пожаловать!");

// Вывод приветственного сообщения в окно браузера
document.write("<p style='color: green; font-weight: bold;'>Сообщение из внешнего файла: Рады видеть вас!</p>");
```

## Задание 3: Ex3.html

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Задание 3 - Взаимодействие с пользователем</title>
</head>
<body>
    <h1>Задание 3</h1>
    
    <script>
        // Запрос имени у пользователя с значением по умолчанию
        let userName = prompt("Введите сюда своё имя", "Введите имя");
        
        // Проверка на отмену ввода
        if (userName === null || userName === "" || userName === "Введите имя") {
            userName = "неизвестный пользователь";
        }
        
        // Вывод имени в окно браузера
        document.write("<p style='font-size: 18px; color: red;'>Ваше имя <strong>" + userName + "</strong></p>");
        
        // Диалоговое окно подтверждения
        let restart = confirm("Начать заново?");
        
        // Обработка ответа пользователя
        if (restart) {
            alert("Не надоело?");
        } else {
            alert("Ну и правильно!");
        }
    </script>
    
    <p>Спасибо за участие в опросе!</p>
</body>
</html>
```

## Инструкция по сохранению файлов:

2. **Для Задания 2:**
   - Сохраните HTML код как `Ex2.html`

   - **Важно:** При сохранении `main.js` выберите:
     - Тип файла: "Все файлы"
     - Кодировка: "UTF-8"

3. **Для Задания 3:** Сохраните код как `Ex3.html`

## Особенности реализации:

**Задание 1:**
- Скрипт расположен между двумя абзацами
- Используется `alert()` для диалогового окна
- Используется `document.write()` для вывода в браузер

**Задание 2:**
- JavaScript код вынесен в отдельный файл
- Подключение через `<script src="main.js"></script>`
- Файл сохраняется с кодировкой UTF-8

**Задание 3:**
- Используется `prompt()` для ввода имени
- Переменная `userName` хранит введенное значение
- `document.write()` выводит имя на страницу
- `confirm()` создает диалог подтверждения
- `alert()` показывает разные сообщения в зависимости от выбора

Все файлы должны находиться в одной рабочей папке для корректной работы.

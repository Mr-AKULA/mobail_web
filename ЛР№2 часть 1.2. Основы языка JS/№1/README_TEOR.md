# Практическая работа №1. Размещение скриптов в HTML-документе

В этой практической работе рассматриваются различные способы подключения JavaScript-кода к HTML-документу.

## Задание 1: Встроенный скрипт (Inline Script)

**Цель:** Научиться добавлять JavaScript-код непосредственно в HTML-документ с помощью тега `<script>`.

**Файлы:**
- `Ex1.html`

**Задачи:**
1. Создать простой HTML-документ.
2. Добавить два абзаца (`<p>`) с произвольным текстом.
3. Между абзацами разместить тег `<script>` с кодом, который выводит приветственное сообщение в диалоговом окне `alert()`.
4. После закрытия диалогового окна, вывести аналогичное приветствие непосредственно в тело HTML-документа с помощью `document.write()`.

**Пример ожидаемого результата:**
При открытии `Ex1.html` сначала появляется окно с приветствием, а после его закрытия текст приветствия добавляется на страницу между двумя абзацами.

## Задание 3: Взаимодействие с пользователем

**Цель:** Изучить базовые методы взаимодействия с пользователем: `prompt()`, `alert()` и `confirm()`.

**Файлы:**
- `Ex3.html`

**Задачи:**
1. Создать простой HTML-документ.
2. Использовать метод `prompt()`, чтобы отобразить диалоговое окно с предложением ввести имя и значением по умолчанию.
3. Сохранить введенное пользователем значение в переменную.
4. Вывести на страницу (в окно браузера) строку: "Ваше имя `<введенное_имя>`".
5. Использовать метод `confirm()`, чтобы спросить пользователя "Начать заново?".
   - При положительном ответе (`true`) показать сообщение `alert("Не надоело?")`.
   - При отрицательном ответе (`false`) показать сообщение `alert("Ну и правильно!")`.

**Пример реализации кода в `Ex3.html`:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>Задание 3</title>
</head>
<body>
    <script>
        // Запрос имени у пользователя
        let userName = prompt('Введите сюда своё имя', 'Введите имя');

        // Вывод имени на страницу
        document.write(`<p>Ваше имя <strong>${userName}</strong></p>`);

        // Диалог подтверждения
        let shouldRestart = confirm('Начать заново?');

        if (shouldRestart) {
            alert('Не надоело?');
        } else {
            alert('Ну и правильно!');
        }
    </script>
</body>
</html>
```

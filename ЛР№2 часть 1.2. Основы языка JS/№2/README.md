## Задание 4: Условные операторы

**Цель:** Научиться работать с условными операторами `if-else` и организовывать проверку условий.

**Файлы:**
- `Ex4.html`

**Задачи:**
1. Модифицировать предоставленный скрипт для обработки случая, когда введенные числа равны.
2. Создать скрипт проверки пароля с выводом соответствующего сообщения.

**Пример реализации для сравнения чисел:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>Сравнение чисел</title>
</head>
<body>
    <script>
        var x, y;
        x = parseInt(prompt("Введите значение х", ''));
        y = parseInt(prompt("Введите значение у", ''));
        
        if (x < y) {
            alert("Максимальное число - y");
        } else if (x > y) {
            alert("Максимальное число - x");
        } else {
            alert("Введенные числа равны!");
        }
    </script>
</body>
</html>
```

**Пример реализации проверки пароля:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>Проверка пароля</title>
</head>
<body>
    <script>
        var correctPassword = "12345"; // Верный пароль
        var userPassword = prompt("Введите пароль:", "");
        
        if (userPassword === correctPassword) {
            document.write("<p style='color: green;'>Пароль верен! Доступ разрешен.</p>");
        } else {
            document.write("<p style='color: red;'>Неверный пароль! Доступ запрещен.</p>");
        }
    </script>
</body>
</html>
```

## Задание 5: Циклы и функции

**Цель:** Научиться создавать функции и работать с циклами `for`.

**Файлы:**
- `Ex5.html`

**Задачи:**
1. Создать функцию для рисования линий.
2. Использовать цикл `for` для отрисовки 10 линий, где каждая последующая длиннее предыдущей на 10 пикселей.

**Пример реализации:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>Рисование линий</title>
</head>
<body>
    <h1>Линии увеличивающейся длины</h1>
    <script>
        function drawLine(width) {
            document.writeln("<hr align='left' width='" + width + "' style='margin: 10px 0;'>");
        }
        
        for (var i = 1; i <= 10; i++) {
            drawLine(i * 10);
        }
    </script>
</body>
</html>
```

## Задание 6: Таблица степеней двойки

**Цель:** Научиться использовать математические операции и формировать таблицы с помощью JavaScript.

**Файлы:**
- `Ex6.html`

**Задачи:**
1. Создать таблицу степеней двойки от 2⁰ до 2⁵.
2. Использовать метод `Math.pow()` для вычисления степеней.
3. Использовать метод `document.write()` для формирования таблицы.

**Пример реализации:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>Таблица степеней двойки</title>
    <style>
        table {
            border-collapse: collapse;
            margin: 20px;
        }
        th, td {
            border: 1px solid #000;
            padding: 8px 15px;
            text-align: center;
        }
        th {
            background-color: #f2f2f2;
        }
    </style>
</head>
<body>
    <script>
        document.write("<table>");
        document.write("<tr><th>Степень</th><th>Результат</th></tr>");
        
        for (var i = 0; i <= 5; i++) {
            var result = Math.pow(2, i);
            document.write("<tr>");
            document.write("<td>2<sup>" + i + "</sup></td>");
            document.write("<td>" + result + "</td>");
            document.write("</tr>");
        }
        
        document.write("</table>");
    </script>
</body>
</html>
```

## Задание 7: Работа с массивами

**Цель:** Научиться работать с массивами и методом `slice()`.

**Файлы:**
- `Ex7.html`

**Задачи:**
1. Создать массив школьных предметов.
2. Разделить предметы на точные и гуманитарные науки.
3. Использовать функцию для организации кода.

**Пример реализации:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>Школьные предметы</title>
    <script>
        function displaySubjects() {
            var subjects = ["Математика", "Физика", "Химия", "Информатика", 
                           "История", "Литература", "Русский язык", "Обществознание", 
                           "Биология", "География", "Английский язык"];
            
            // Точные науки (первые 4 предмета)
            var exactSciences = subjects.slice(0, 4);
            
            // Гуманитарные науки (предметы с 4 по 7 индекс)
            var humanities = subjects.slice(4, 8);
            
            document.write("<h2>Точные науки:</h2>");
            document.write("<p>" + exactSciences.join(", ") + "</p>");
            
            document.write("<h2>Гуманитарные науки:</h2>");
            document.write("<p>" + humanities.join(", ") + "</p>");
        }
    </script>
</head>
<body>
    <h1>Классификация школьных предметов</h1>
    <script>
        displaySubjects();
    </script>
</body>
</html>
```

## Задание 8: Определение браузера и перенаправление

**Цель:** Научиться определять тип браузера и выполнять перенаправление.

**Файлы:**
- `Ex8.html`

**Задачи:**
1. Определить браузер пользователя с помощью свойства `navigator.appName`.
2. Выполнить перенаправление на соответствующую страницу.

**Пример реализации:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>Определение браузера</title>
</head>
<body>
    <script>
        var browserName = navigator.appName;
        
        if (browserName === "Microsoft Internet Explorer") {
            alert("Определен Internet Explorer. Перенаправляем на Ex1.html");
            window.location.href = "Ex1.html";
        } else {
            alert("Определен другой браузер: " + browserName + ". Перенаправляем на Ex3.html");
            window.location.href = "Ex3.html";
        }
    </script>
</body>
</html>
```

## Примечания

- Для Задания 8 убедитесь, что файлы `Ex1.html` и `Ex3.html` существуют в той же папке.
- Метод `Math.pow(base, exponent)` возвращает основание, возведенное в степень показатель.
- Метод `array.slice(start, end)` возвращает часть массива от индекса `start` до `end` (не включая `end`).
- Свойство `navigator.appName` возвращает название браузера.

ЛР№2 часть 3. Разработка приложения на основе стека языков HTML, CSS, JS
Задание:
Необходимо разработать мобильное приложение, используя HTML и CSS для создания интерфейса и JS для создания функционала, способное рассчитывать периметр, площадь и объём для таких геометрических фигур как: квадрат, прямоугольник, треугольник, параллелограмм, трапеция, ромб, круг, куб, параллелепипед, конус, пирамида, цилиндр, шар.

Требования:
1. Необходимо обеспечить ввод таких параметров фигур как: размеры сторон, высота, радиус.
2. Необходимо использовать соответствующие элементы управления.
3. Приложение должно быть удобным, понятным и дружелюбным.
4. GUI должен быть оформлен в выбранной Вами цветовой гамме и стиле, сопровождаться поясняющими изображениями.
5. Интерфейс должен быть динамичным. Например, при выборе фигуры для ввода должны быть доступны только соответствующие фигуре параметры.
6. Приложение должно иметь понятное название и логотип.
7. Необходимо проверять введенные пользователем данные и выдавать соответствующий результат.
8. Результат работы программы должен отображаться на экране.

Разработанные проект и приложение необходимо продемонстрировать преподавателю на занятиях.

Для тестирования и отладки можете использовать любые доступные Вам ресурсы.

Ожидаемые различия между Internet Explorer и Firefox:

  • Internet Explorer (старые версии):
    • Текст может выходить за границы контейнера
    • Отсутствие автоматической прокрутки
    • Проблемы с отображением свойства overflow
    • Неправильный расчет высоты с учетом padding
  • Firefox (и другие современные браузеры):
    • Корректное отображение полосы прокрутки
    • Текст остается внутри заданных границ
    • Правильная работа свойства overflow: auto
    • Точное соблюдение размеров 300×100px
